## 🌟 Key Features

*   **Premium Welcome Screen:** A cinematic sci-fi entrance screen featuring an HSL-glowing text-fill effect, cosmic rotating background pattern, and a smooth slide-up fade out (2.5s timeout) optimized for browser performance.
*   **Interactive Hero Section:** Utilizes custom-configured **Particles.js** background network lines, dynamic float shapes, and smooth magnetic mouse-reactive button offsets.
*   **Fluid Scroll Animations:** Rich entrance effects powered by **GSAP (GreenSock) & ScrollTrigger** alongside **ScrollReveal** for an organic feel as users browse.
*   **3D Interactive Hover (Glassmorphism):** High-end 3D tilt interactions built via **Vanilla-Tilt.js** integrated on project cards, stats, and profile images, complete with a subtle reflection glare.
*   **Intersection Observer Stats Counters:** Numerical accomplishments that automatically count up dynamically only when scrolled into the viewport.
*   **Modern Branding Grid:**
    *   **Personal Info Grid:** A beautifully structured overview featuring degree status, current freelancing availability, location, and quick contact details.
    *   **Publications & Freelance Experience:** Dedicated section highlighting professional freelance web development services and bespoke Medium/Substack travel and copywriting article previews with responsive external redirection.
*   **Zero-Overhead Responsive Layout:** Coded with vanilla CSS Grid and Flexbox, fully optimized for seamless viewing on screens down to 360px (mobile viewports).

    *   [GSAP](https://greensock.com/gsap/) (High-performance UI animations)

    *   [Vanilla-Tilt](https://micku7zu.github.io/vanilla-tilt.js/) (Fluid 3D hover effects)
    *   [FontAwesome v6](https://fontawesome.com/) (Modern high-resolution vector iconography)

## 🚀 Installation & Local Development

This portfolio is built as a static application, meaning it has zero server overhead and works directly in any browser environment!

### **Option 1: Open Locally**
Simply double-click `index.html` to open it in your web browser of choice.

### **Option 2: Using Local Live Server**
1. If you are using VS Code, install the **Live Server** extension.
2. Click **"Go Live"** in the bottom right corner of the editor to run a hot-reloading development server on `http://127.0.0.1:5500`.

### **Option 3: Quick Terminal Host**
You can also run a quick python server from the project directory:
```bash
# Python 3
python -m http.server 8000
```
Then visit `http://localhost:8000`.

## 🌎 Deployment to Vercel

To deploy your portfolio to production via **Vercel CLI**:

1. Install Vercel CLI globally or run it on the fly:

2. Log in using your GitHub account and follow the automatic setup prompts.
3. Your site will be online globally in under 10 seconds!
